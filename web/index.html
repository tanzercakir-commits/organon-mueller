organon-mueller — result viewer

Stokes–Mueller decomposition results, rendered in your browser.

This page does no computation and needs no server. The maths lives in the Python package (organon-mueller); run a tool via the MCP server or CLI, then paste its JSON output below to view it. Nothing you paste leaves your machine — the page has no network access and never runs pasted data as code.

Evidence labels map to the verification layers in docs/VERIFICATION.md: symbolic-proof (exact), numeric-deterministic (seeded), candidate (no novelty or physics claim — that judgement is reserved for humans). This viewer is presentation only; it is not hosted anywhere by the project.
